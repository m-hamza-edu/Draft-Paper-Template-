# 📄 Draft Paper Template

A minimalist, production-ready LaTeX template for academic writing with a clean structure, pre-configured packages, and seamless Overleaf support.

# ✨ Features

- ✅ Fully Overleaf Compatible  
- ✅ Clean Academic Paper Structure  
- ✅ Pre-configured LaTeX Packages  
- ✅ Ready for Figures, Tables, and Algorithms  
- ✅ Bibliography Support (`biblatex`)  
- ✅ Two-Column Conference Style Layout  
- ✅ Lightweight and Easy to Customize  

---

# 📂 Template Contents

The project includes:

- `main.tex` — Main paper file  
- `references.bib` — Bibliography database  
- `figures/` — Images and plots  
- `tables/` — Table resources  
- Predefined formatting and package setup
